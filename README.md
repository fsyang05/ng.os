# ng.os

An AI-native operating system for nonprofits, live with organizations in Baltimore.

## Stack

Made using Turborepo.
Webapp built with React and TypeScript, server built with Python. Voice agents hosted on Vapi.

Note: this repository contains a demo, not the actual product

Voice agent demonstration available [here](https://drive.google.com/file/d/1cP3VuqKThVdj6L2bO8WBSdaNET0MRnym/view?usp=sharing)
